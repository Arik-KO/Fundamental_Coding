# Array Related Problems

This folder contains solutions to some array related problems. It contains merge of two sorted arrays, rotation of an array, use of sliding window and finding duplicate elements. All the problems are solved here using Python programming language.

- To find duplicate, I have used naive approach and also, set() data structure of Python
- To find rotation, I have made general form that checks how many elements to go through rotation for an array and in which direction.
- To merge two sorted array, I have used to pointers to keep track of the smallest element.
- Finally, for sliding window, used naive solution to store the maximum element from each window.

### Note : For data hinting, from typing -> List, has been imported but data hinting is not mandatory. Therefore, it is upto the user how they want to use them in the scripts.
